# Magdeburger Halbkugeln – Web-App v3

Statische Web-App für GitHub Pages.

## Dateien

- `index.html` – komplette Single-Page-App
- `styles.css` – Layout, Animationen, responsive Design
- `script.js` – Navigation und Simulation
- `assets/startfoto.jpg` – Ausstellungsfoto der Startseite
- `assets/pumpe_detail.jpg` – Detailfoto der historischen Pumpe
- `assets/ovgu_logo.png` – OvGU-Logo im Header

## Veröffentlichung über GitHub Pages

1. Ordner in ein GitHub-Repository hochladen.
2. In den Repository-Einstellungen `Pages` aktivieren.
3. Den Branch mit diesem Projektordner auswählen.
4. `index.html` im Hauptverzeichnis belassen.

## Schnell anpassbar

- Otto-Sprechertext: `index.html` im Bereich `ottoPage`
- Physikparameter: `script.js` am Dateianfang
- Hotspots der Startseite: `styles.css` bei `.hotspot-*`
